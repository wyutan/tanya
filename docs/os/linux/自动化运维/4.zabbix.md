## 准备工作
### 安装`net-snmp-utils`

- Ubuntu/Debian
```
apt-get install snmp snmp-mibs-downloader
```
- CentOS/RHEL
```
yum install net-snmp-utils
```
### 修改配置文件
编辑`/etc/snmp/snmp.conf`，添加以下行让系统加载所有 MIB：
```
mibs +ALL
```
### 重启zabbix服务
```
systemctl restart zabbix-server
```
### 将mib文件上传至服务器
将下载好的mib文件上传至zabbix服务器的`/usr/share/snmp/mibs`目录下
