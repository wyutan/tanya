### 常见路由协议管理距离 (AD) 对照表

| 路由来源 | 管理距离值 (Administrative Distance) |
| :--- | :---: |
| 连接路由 (Connected) | 0 |
| 下一跳为路由器接口的静态路由 | 1 |
| 静态路由 (Static Route) | 1 |
| EIGRP 汇总路由 | 5 |
| 外部 BGP (eBGP) | 20 |
| 内部 EIGRP | 90 |
| IGRP | 100 |
| OSPF | 110 |
| IS-IS | 115 |
| RIP-1 / RIP-2 | 120 |
| EGP | 140 |
| 外部 EIGRP | 170 |
| 内部 BGP (iBGP) | 200 |
| 未知 (Unknown) | 255 |

> **提示**：管理距离（AD）用于衡量路由信息源的可信度。数值越小，优先级越高。
